## ¿Qué es este repositorio?

Este repositorio es tu **Trabajo Práctico Individual (TPI)** de la materia. Contiene una serie de ejercicios progresivos que cubren HTML, CSS y JavaScript, organizados en dos módulos de dificultad creciente.

> **Nivel de dificultad:** Va desde ejercicios para principiantes absolutos hasta desafíos de nivel profesional senior.

## 🚀 Cómo empezar

### 1. Hacé un fork de este repositorio
```bash
# En GitHub: click en el botón "Fork" (arriba a la derecha)
# Luego cloná TU fork (no el original):
git clone https://github.com/TU-USUARIO/tp-frontend.git
cd tp-frontend
```

### 2. Creá tu rama de trabajo
```bash
git checkout -b tp/APELLIDO-NOMBRE
# Ejemplo: git checkout -b tp/garcia-maria
```

### 3. Completá los ejercicios
Cada ejercicio tiene su propia carpeta con un archivo `SPEC.md` que describe los requerimientos en formato estándar.

### 4. Entregá con un Pull Request
Al terminar cada módulo, abrí un **Pull Request** hacia el repositorio original con el template provisto.

## ⚠️ Política de uso de Inteligencia Artificial

Este TP **permite y reconoce** el uso de herramientas de IA generativa (ChatGPT, Claude, Copilot, etc.) como parte del proceso de aprendizaje. Sin embargo, **el uso de IA sin comprensión es trampa**.

Cada ejercicio tiene un archivo `REFLEXION.md` obligatorio donde debés:
1. Explicar con tus palabras qué hace tu solución
2. Responder preguntas conceptuales específicas
3. Documentar qué hiciste con/sin ayuda de IA
4. Defender decisiones técnicas tomadas

> 💡 **Regla de oro:** Si no podés explicar tu código en una defensa oral de 5 minutos, no aprobás el ejercicio — sin importar que esté funcionando.

Leé `CODIGO_DE_HONOR.md` antes de comenzar.
